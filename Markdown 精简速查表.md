## 文本样式
- 标题：`# H1` `## H2` `### H3` （最多 6 级）
- 强调：`*斜体*` 或 `_斜体_`  
- 加粗：`**粗体**` 或 `__粗体__`  
- 删掉线：`~~删除线~~`  
- 行内代码：`` `代码` ``

## 段落与换行
- 段落：空行分隔段落。
- 强制换行：行尾加两个空格然后回车。

## 列表
- 无序列表示例：
  - `- 项目` 或 `* 项目` 或 `+ 项目`
- 有序列表示例：
  - `1. 第一` `2. 第二`
- 嵌套列表：缩进两个或四个空格

## 链接与图片
- 行内链接：`[链接文本](https://example.com "可选标题")`
- 自动链接：`<https://example.com>`
- 图片：`![替代文本](路径或URL "可选标题")`

## 代码块
- 行内：`` `代码` ``
- 多行代码块（带语言高亮）：  ````
```python
def hello():
  print("Hello Markdown")
```
